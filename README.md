# vue-cnode [预览网址](https://jiangwenzhe.github.io/vue-cnode/#/)

**一个仿cnode的项目：技术栈为 `Vue + Vuex + Vue Router`**

## Project setup

```bash
yarn install
```

### Compiles and hot-reloads for development

```bash
yarn run serve
```

### Compiles and minifies for production

```bash
yarn run build
```

### Run your tests

```bash
yarn run test
```

### Lints and fixes files

```bash
yarn run lint
```
